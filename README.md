# Campus Meal Budget Planner

Campus Meal Budget Planner is a simple web service that helps university
students plan daily or weekly meal spending. It calculates an affordable
per-meal target from the user's remaining budget and presents a practical meal
strategy with three locally stored meal suggestions.

This project was built for Introduction to AI Programming Assignment 4.

## Main Features

- Three clearly navigable pages: Home, Planner, and About
- Interactive meal budget calculator
- Remaining balance and recommended per-meal budget
- Safe, caution, and over-budget status feedback
- Meal strategy based on the calculated budget
- Three meal suggestions ranked by budget and preferred style
- Input validation and reset functionality
- Responsive, student-friendly interface
- No backend, database, account, or external API

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- Local TypeScript meal suggestion data

## Run Locally

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in a browser.

## Build

Create a production build:

```bash
npm run build
```

Run the production build locally:

```bash
npm start
```
